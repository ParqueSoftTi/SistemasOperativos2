1. Crea un directorio llamado practica_archivos en tu directorio personal.
  2. Dentro de practica_archivos, crea tres archivos: archivo1.txt, archivo2.txt y archivo3.txt.
  3. Utiliza el comando ls para listar el contenido del directorio y asegúrate de que los archivos se hayan creado correctamente.
  4. Cambia el nombre del archivo archivo3.txt a nuevo_archivo.txt.
  5. Mueve el archivo nuevo_archivo.txt al directorio padre (practica_archivos) utilizando el comando mv.
  6. Elimina el archivo archivo1.txt utilizando el comando rm.
  7. Verifica que el archivo haya sido eliminado utilizando ls.
  8. Finalmente, borra el directorio practica_archivos y su contenido.

Los comandos que utilizaremos en esta practica son:

mkdir ~/practica_archivos
touch ~/practica_archivos/archivo1.txt ~/practica_archivos/archivo2.txt ~/practica_archivos/archivo3.txt
ls ~/practica_archivos
mv ~/practica_archivos/archivo3.txt ~/practica_archivos/nuevo_archivo.txt
mv ~/practica_archivos/nuevo_archivo.txt ~/practica_archivos/..
rm ~/practica_archivos/archivo1.txt
ls ~/practica_archivos
rm -r ~/practica_archivos
